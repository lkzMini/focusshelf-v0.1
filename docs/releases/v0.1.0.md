# FocusShelf v0.1.0

## Summary

FocusShelf v0.1.0 is the first public early-MVP release of a small local-first focus shelf for Windows.

The app keeps a deliberately tiny scope:

- one current focus
- one next step
- a quick note
- a simple timer
- a small done list
- a daily reminder

## Packaging note

This release uses a **Release ZIP** instead of self-signed MSIX.

Self-signed MSIX was rejected as the default public packaging path because it asks users to trust/install a certificate before running a small MVP app. That is poor UX and a bad security posture for a public release.

The previous self-contained `dotnet publish` ZIP also failed the clean-folder smoke test with:

- faulting module: `Microsoft.UI.Xaml.dll`
- exception code: `0xc000027b`

So v0.1.0 uses the working framework-dependent unpackaged Windows App SDK output style, produced from **Release**, zipped reproducibly, extracted into a clean folder, and smoke-tested before publishing.

## Included in this release

- Local JSON persistence
- Safer state loading and normalization
- Theme-aware readability improvements
- Dark/light text-input contrast fixes
- Cleaner resize behavior via a protected minimum content width and horizontal scroll fallback
- Reproducible Release ZIP script
- Clean-folder launch smoke test in the release script

## GitHub Release artifact

Expected v0.1.0 x64 artifact:

- `FocusShelf-v0.1.0-win-x64.zip`

## Install / run

1. Download `FocusShelf-v0.1.0-win-x64.zip`.
2. Extract it to a normal user-writable folder.
3. Run `FocusShelf.App.exe`.

No admin rights, certificate installation, or MSIX trust step is required.

## Known limitations

- Early MVP only
- No sync, notifications, or accounts
- No advanced task/project management
- ZIP release uses framework-dependent unpackaged Windows App SDK deployment
- MSIX may return later only with a trusted signing story

## Upgrade / compatibility note

State remains stored at:

- `%LOCALAPPDATA%\FocusShelf\state.json`

No intentional state-format migration was introduced in this pass.
